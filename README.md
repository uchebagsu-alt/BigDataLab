# BigDataLab - Обучение

Данный проект является учебным в рамках курса Big Data Lab.

## Цель проекта

Освоить базовые навыки работы с системой контроля версий Git и платформой GitHub, а также научиться создавать документацию для проектов в формате README.md.

## Источники данных

- **ITS LMS** — учебная платформа, предоставляющая материалы и задания по курсу Big Data Lab
- **GitHub** — платформа для хранения и управления кодом проекта
- **Markdown** — язык разметки для оформления документации


## В рамках задания выполнено:

- Установка Git на компьютер
- Создание учетной записи на GitHub
- Создание репозитория на GitHub
- Клонирование репозитория на локальный компьютер
- Создание новой ветки `feature/add_README`
- Создание и заполнение файла README.md
- Отправка изменений в удаленный репозиторий

## Используемые технологии

- Git
- GitHub
- Visual Studio Code
- Markdown
