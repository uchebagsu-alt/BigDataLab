# Классификация CSV-файлов и сущностей
| Файл | Сущность | Тип | Описание |
| :---: | :---: | :---: | :--- |
| sales.csv | sales | Таблица Фактов| История продаж |
| shops.csv | shops | Справочник | Список существующих магазинов |
| categories.csv | categories | Справочник  | Категории товаров |
| cities.csv | cities | Справочник | Список городов,в которых расположены магазины |
| countries.csv | countries | Справочник | Страны, в которых расположены города |
| customers.csv | customers | Справочник | Список клиентов, которые пользовались услугами магазинов|
| employees.csv | employees | Справочник| Список действующих работников магазинов |
| products.cvs | products | Справочник | Каталог товаров |
