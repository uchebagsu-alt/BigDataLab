# <center>Описание предметной области «Магазин продуктов»</center>

&emsp;«Магазин продуктов» представляет собой систему.Её цель учёт продаж, работников, покупателей и склада товаров. Основные сущности которые учавствуют в процессе продаж:

- **<a id='sales'>sales</a>** - хранит информацию о истории продаж, содержит  уникальный айди продажи, айди работника, клиента и продукта, количество, скидка, итоговая цена, метку времени продажи, номер транзакции.

- **<a id='products'>products</a>** - хранит информацию о товарах магазинов, содержит уникальный айди продукта, наименования продуктов, стоимость, категория(such as fruits, vegetables), класс, метку времени изменения, устойчивость, есть ли аллергены.

- **<a id='employees'>employees</a>** - хранит информацию о работниках, содержит  уникальный айди работника, ФИО работника, дата рождения, гендер, айди города, айди магазина в котором работает, дата найма.

- **<a id='customers'>customers</a>** - хранит информацию о клиентах, содержит уникальный айди клиента, ФИО клиента, айди города, адресс.

## Операции

- **Фиксация продаж** 
- **Учет состава сотрудников**
- **Учет клиентов**
- **Обновление ассортимента** 
- **Расширение сети магазинов** 

## Ключевые атрибуты products и employess
<small>*Примечание: при наведении мыши на некоторые слова появится подсказка.*</small>  

&emsp;**[products](#products)**:  
1) *product_name* - <span title="Какой именно товар"> имя товара </span>

2) *price* - <span title="Сколько стоит данный товар"> цена товара </span> 

3) *category_id* - <span title="Какой тип товара(например, фрукт, овощ)"> категория товара </span> 

4) *vitality_days* - <span title="Как долго можем продавать данный товар"> срок годности </span>

&emsp;**[employees](#employees)**:  
1) *first_name* и *last_name* - <span title="Какой именно сотрудник">фамилия и имя сотрудника</span>

2) *city_id* - <span title="Город в котором сотрудник работает">id города</span>

3) *shop_id* - <span title="В каком именно магазине сотрудник работает">id магазина</span>

4) *hire_date* - <span title="Как давно сотрудник работает">дата найма</span>
